**製作人**：霓子（UX-05）
**版本**：v1.0｜2026-04-10

## 視覺參考清單（10 件作品）

### 1. Disco Elysium（遊戲）
**具體可學的元素**
- 對話框的文字排版：左對齊、行距寬鬆、沒有泡泡框、純文字就有份量
- 技能名稱的斜體顯示方式——用字體本身的重量傳遞情緒，不靠色彩
- 失敗選項的「灰色」處理：不用紅色警告，只是淡掉，靜靜地告訴你「你沒有資格選這個」

### 2. Citizen Sleeper（遊戲）
**具體可學的元素**
- 狀態面板（EV/SRR 這類數值）的視覺化方式：六角形骰子格 + 溫和的暖色 icon
- 「日記式」的文字介面：不是對話框，而是像是有人在記錄的頁面感
- 角色插圖的局部裁切——只有臉或手出現在邊緣，比全身立繪更有距離感也更神秘

### 3. Slay the Princess（遊戲）
**具體可學的元素**
- 黑白手繪線條配上少量飽和度極低的彩色：用最少的顏色做出最強的視覺衝擊
- 選項文字的視覺排版：每個選項都是一行，行與行之間的間距本身就在製造呼吸感
- 字體替換的技巧：同一段文字，不同聲音用不同字型，不需要頭像就能區分角色

### 4. Heaven's Vault（遊戲）
**具體可學的元素**
- 「線索收集」的 UI：像是翻閱筆記本的觸感，每條線索有手寫感的標記
- 地圖介面的設計：不是俯視地圖，而是場景縮圖的連結，更有探索感
- 字體選擇：Libre Baskerville 類的有衯線字體用在古典感的場景，現代感場景用無衯線

### 5. 文字遊戲《Fallen London》（網頁遊戲）
**具體可學的元素**
- 純文字敘事的節奏控制：每一段不超過三句，段落之間用空行呼吸
- 「選項說明」的文字設計：選項本身一行，下方用淡色小字補充後果提示
- 暗色網頁背景上的文字對比：主文用 `#e8dcc8`（溫暖米白），不是純白，護眼又有紙質感

### 6. 台灣獨立遊戲《怪獸遠征》的選關畫面（遊戲）
**具體可學的元素**
- 台灣設計師處理中文字體的方式：Noto Serif TC 在小字號的間距調整（letter-spacing: 0.05em）
- 選單的簡約排版：中文不需要多餘的裝飾，字本身就是視覺元素

### 7. 網頁作品《Bruno Simon Portfolio》（https://bruno-simon.com）
**具體可學的元素**
- 網頁字體的使用：monospace 字體搭配文字動效有獨特的「系統感」
- 載入動畫的設計語言：不用旋轉圓圈，用文字本身來做動態

### 8. The House in Fata Morgana（視覺小說）
**具體可學的元素**
- 過場畫面的背景處理：靜態插圖配合緩慢的視差滾動，比純靜態圖有空間感
- 結局畫面的文字排版：標題大、說明小、大量留白，讓玩家自己坐在那個重量裡

### 9. 網頁設計作品《Linear App》的暗色 UI（https://linear.app）
**具體可學的元素**
- 專業暗色系 UI 的配色邏輯：背景不是純黑 `#000000`，而是有藍調的深色 `#0f1117`
- 邊框和分隔線的顏色：不用白色，用 `rgba(255,255,255,0.08)` 這類低透明度白，精緻很多
- 懸停效果：`background-color` 不改顏色，只提亮 5%，比直接換色高級

### 10. Noir Syndrome（遊戲 UI 參考）
**具體可學的元素**
- 「線索板」的視覺設計：照片、字條、紅線連接，混搭不同媒材的質感
- 字體降解效果：打字機不完整印刷的感覺，可以用 CSS `font-variation-settings` 或特定字體實現

## 賽德里斯配色候選

> 設計原則：賽德里斯是一座「繁華霓虹與鋼鐵森林掩蓋下充滿監控與陰影的港口都市」。
> 配色要能同時承載「霓虹繁華」和「陰影監控」，不能只選其中一個方向。

---

### 方案 A：鐵鏽港夜晚（霓子第一推薦）

```
背景色　　#0d1117    深藍黑，有輕微藍調，不是純黑
主文字　　#e2c9a0    溫暖米白，帶輕微黃調，有紙質感
強調色　　#ff3864    霓虹玫瑰紅，用在 NPC 名字、重要線索、危險提示
次強調色　#5bb8ff    冷藍，用在 BTMA 相關 UI、費洛蒙掃描儀元素
危險色　　#ff6b35    橙紅，用在 EV 值過高警告、死亡提示
介面灰　　#2d3748    深灰，用在卡片背景、輸入框背景
邊框色　　rgba(255,255,255,0.08)    低透明度白，精緻邊框
```

**為什麼推薦**：`#ff3864` 的玫瑰紅比純紅少了一分暴力感，多了一分「這個城市裡有些美麗是危險的」的曖昧感。冷藍強調色讓 BTMA 的科技感有明確的視覺區分。

---

### 方案 B：原初進化教的神秘感

```
背景色　　#1a1025    深紫黑，帶輕微紫調
主文字　　#d4c5e2    淡紫白
強調色　　#c084fc    螢光紫，用在教派相關元素
次強調色　#f472b6    粉紅，用在情感高峰場景
危險色　　#ef4444    標準紅
介面灰　　#2e1f40    深紫灰
邊框色　　rgba(192,132,252,0.15)    低透明紫
```

**適用情境**：如果想讓「原初進化教」的場景有明確的視覺區隔，這套可以作為「教派場景專用主題」，和方案 A 搭配使用。

---

### 方案 C：上城區聖格瑞斯的冷靜科技感

```
背景色　　#0a0f1e    深海軍藍黑
主文字　　#cdd6f4    冷白藍，接近月光感
強調色　　#89b4fa    藍色，BTMA 基調色
次強調色　#a6e3a1    冷綠，系統正常提示
危險色　　#f38ba8    冷粉紅，比純紅更有「系統警告」感
介面灰　　#1e2030    深藍灰
邊框色　　rgba(137,180,250,0.12)
```

**適用情境**：以第二相體路線（Route B）為主的玩法，或者 BTMA 系統 UI 的專用主題。

## 霓子的最終建議

**主配色選方案 A，方案 B 和 C 作為場景切換用的子主題。**

Phase 5 的工作分配：
- 主 UI（對話框、選單、成就）→ 方案 A
- 教派相關場景、第六幕「連環」段落 → 方案 B 特定元素
- 警局、BTMA 審訊場景 → 方案 C 特定元素

**字體最終決定**（等 Phase 5 正式確認，但這是我的強烈建議）：
- NPC 對話、線索文字、敘事段落 → Noto Serif TC（weight: 400 / 700）
- UI 元素、按鈕文字、數值 → Inter（weight: 400 / 500 / 700）
- 系統訊息、代碼感文字 → JetBrains Mono（weight: 400）

三個都是 Google Fonts 免費，可以直接用 `next/font/google` 引入。
