## 各兇手詳細說明

### 韓卓（法醫助理）
**唯一有效動機**：方向 B（滅口者）
- 他的背景是「那晚被叫去處理現場，看見了不該看見的東西，被要求簽保密協議」，符合滅口者的「奉命清除知情目擊者」邏輯
- 配合元素 ⑩（玩家身邊的人主動消失是在保護玩家）效果最強
- **隨機時**：確認是方向 B 才選韓卓，否則重抽

### 余霜（兒科護士）
**唯一有效動機**：方向 D（扭曲崇拜）
- 她的背景是「對林淵的情感早已超出正常護患關係」，她相信失蹤者是「不配見證林淵覺醒的人」
- 配合元素 ⑥（兇手和玩家認識很久）或元素 ⑫（玩家是工具）效果最強
- **隨機時**：確認是方向 D 才選余霜

### 鄭博（保險理賠員）
**唯一有效動機**：方向 A（倖存者清算）
- 他的妻子是連環案第一個受害者，他自己調查後清算當晚所有相關者
- 配合元素 ⑨（兇手殺了一個他知道是無辜的人）製造道德複雜度
- **特殊設計**：鄭博的調查路線和玩家完全重疊，這是「玩家遇上另一個自己」的鏡像設計
- **隨機時**：確認是方向 A 才選鄭博

### 「它」（身份不明）
**唯一有效動機**：方向 C（實驗失敗品）
- 「它」本身就是那晚出錯的覺醒者，沒有人類意義上的「動機」，只是在找某樣東西
- 配合元素 ① 或 ③（還有其他失蹤者、有人還活著）效果最強
- **特殊設計**：這個兇手全程以「玩家認識的某個普通人」的面貌出現，玩家到接近尾聲才意識到
- **隨機時**：確認是方向 C 才選「它」

### 白秋（藥局藥師）✦ 雙動機角色
**有效動機**：方向 A（可選）或方向 B（可選）
- 方向 A 版本：她的弟弟是那晚的知情者，後來成為受害者，她在清算那些害了弟弟的人
- 方向 B 版本：某勢力利用她的藥學知識來消除知情者（藥物致使失蹤的手法）
- **需要輔助元素**：兩種版本都需要元素 ⑤（林淵的覺醒被人為誘發）來補完邏輯
- **隨機時**：方向 A 或 B 都可選白秋，但要同時抽到元素 ⑤

### 莊河（退休刑警）
**唯一有效動機**：方向 B（滅口者）
- 他知道案件被移交的真相，也知道證據去了哪裡，現在在清除所有可能讓真相重見天日的人
- 最具份量的「沉默」設計：他告訴玩家「你是第幾個來問我的人」——暗示前幾個來問的人都死了，而他是讓他們死的人
- **隨機時**：確認是方向 B，且這局沒有選韓卓（同動機同方向的兩個兇手身份會互相干擾）

### 林知夏（生物系學生）
**唯一有效動機**：方向 D（扭曲崇拜）
- 她崇拜的不是林淵這個人，而是林淵覺醒那一刻代表的意義，用儀式感重現那晚
- 日記裡評語「配不配成為林淵的一部分」是全角色庫最細思極恐的設計之一
- 配合元素 ⑧（監視錄影空白）：她就是那 40 分鐘的關鍵人物
- **隨機時**：確認是方向 D，且這局沒有選余霜（兩個方向 D 角色不共存）

### 陶生（建築工地領班）
**唯一有效動機**：方向 C（實驗失敗品）
- 他本人也是不知情的覺醒者，「兇手」的身份是無意識行為，道德層面是受害者
- 玩家必須自己判斷「這是兇手還是另一個受害者」是這個角色的核心張力
- **特殊設計**：他是整個兇手池裡道德最模糊的角色，適合想要非黑即白以外結局的玩家
- **隨機時**：確認是方向 C，且這局沒有選「它」（兩個方向 C 角色不共存）

## 禁止組合（Forbidden Pairs）

以下組合在同一局裡絕對不能同時出現：

| 組合 | 禁止原因 |
|---|---|
| 余霜 + 林知夏 | 兩個方向 D，角色劇情線重疊，玩家無法分辨哪個才是兇手 |
| 「它」+ 陶生 | 兩個方向 C，且都有「玩家已見過但沒察覺」的設計，過於混亂 |
| 韓卓 + 莊河 | 兩個方向 B，且都有「在幕後默默清除目擊者」的行動模式，線索會互相污染 |
